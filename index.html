BORDAS DO CONHECIMENTO (FAQ)

A: O sistema usa RBAC (Role-Based Access Control). Administradores têm acesso total, moderadores têm acesso parcial e capitães enxergam apenas os dados da sua própria atlética. Cada "classe de herói" tem suas permissões mapeadas no back-end.

A: O back-end diferencia os usuários com base na matrícula. O cargo administrativo é vinculado ao código de matrícula do usuário, garantindo que apenas os coordenadores e membros autorizados acessem painéis restritos do sistema.
A: Sim! O sistema contará com um painel dedicado para download de relatórios de participação de atletas com suas estatísticas, partidas e modalidades. Serão gerados também relatórios de engajamento e impacto social para a Reitoria.
A: O sistema verifica o ID único do atleta antes de registrar qualquer inscrição. Se o mesmo ID já estiver associado àquela modalidade, o cadastro é bloqueado automaticamente, impedindo duplicações e inscrições irregulares.
A: Novos usuários fazem login com sua matrícula tanto no campo de usuário quanto no de senha na primeira vez. Após o primeiro acesso, são redirecionados para uma tela obrigatória de definição de senha segura personalizada.
